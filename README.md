# Zomato Data Analysis Project

This project is based on a Zomato dataset and focuses on analyzing restaurant menu data using Python.

## Tools and Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Tasks
- Cuisine-wise analysis
- Top restaurants analysis
- Price distribution analysis
- Votes analysis
- Dining rating analysis
- City-wise restaurant analysis
- Data visualization using charts

## Project Highlights
- Performed data cleaning and preprocessing
- Converted columns into proper numeric format
- Created bar charts, histograms, pie charts, and scatter plots
- Explored restaurant trends, prices, and customer preferences
